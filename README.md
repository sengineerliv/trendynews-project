A full-stack TypeScript application that delivers personalized news using modern web technologies.

## 🚀 Features

- 🔐 Authentication (JWT)
- 🧠 Personalized news feed
- ⭐ Bookmark articles
- 🔍 Search & filtering
- 📊 Trending analytics
- ⚡ Real-time updates 
- 🤖 AI-powered summaries

## 🛠️ Tech Stack

### Frontend
- Next.js (TypeScript)
- React
- Custom hooks

### Backend
- Node.js + Express (TypeScript)
- MongoDB + Mongoose
- REST API architecture
